# 子智能体边界测试用例

## 一、FileAgent 接口测试用例

### 1. search_file 接口
| 测试场景 | 输入参数 | 预期结果 | 优先级 |
|---------|---------|---------|-------|
| 搜索路径为空字符串 | `search_path=""` | 返回错误："入参错误：search_path必须是非空字符串" | 高 |
| 搜索路径不存在 | `search_path="/不存在的路径"` | 返回错误："搜索路径不存在：/不存在的路径" | 高 |
| 关键词为空字符串 | `keyword=""` | 返回错误："入参错误：keyword必须是非空字符串" | 高 |
| recursive非布尔值 | `recursive="true"` | 返回错误："入参错误：recursive必须是布尔值" | 中 |
| 搜索空目录 | `search_path="/空目录"`, `keyword="test"` | 返回成功："找到0个匹配文件" | 中 |
| 关键词含特殊字符 | `keyword="test*&%"` | 返回匹配结果（无异常） | 中 |
| 递归搜索深度大 | `search_path="/home"`, `keyword="test"`, `recursive=True` | 正常返回所有匹配文件（无超时） | 低 |

### 2. move_to_trash 接口
| 测试场景 | 输入参数 | 预期结果 | 优先级 |
|---------|---------|---------|-------|
| 文件路径为空字符串 | `file_path=""` | 返回错误："入参错误：file_path必须是非空字符串" | 高 |
| 文件不存在 | `file_path="/不存在的文件.txt"` | 返回错误："文件不存在：/不存在的文件.txt" | 高 |
| 文件路径为目录 | `file_path="/home/user1/Desktop"` | 正常移至回收站（目录） | 中 |
| 只读文件 | `file_path="/只读文件.txt"`（chmod 444） | 正常移至回收站 | 中 |
| 系统保护文件 | `file_path="/etc/passwd"` | 返回错误："移动失败：权限不足" | 中 |
| 重复文件名 | 连续移动同名文件 | 文件名自动加后缀（如test.txt → test_1.txt） | 中 |

## 二、SettingsAgent 接口测试用例

### 1. change_wallpaper 接口
| 测试场景 | 输入参数 | 预期结果 | 优先级 |
|---------|---------|---------|-------|
| 壁纸路径为空字符串 | `wallpaper_path=""` | 返回错误："入参错误：wallpaper_path必须是非空字符串" | 高 |
| 壁纸文件不存在 | `wallpaper_path="/不存在的壁纸.jpg"` | 返回错误："壁纸文件不存在：/不存在的壁纸.jpg" | 高 |
| 壁纸格式不支持 | `wallpaper_path="/test.txt"` | 返回错误："仅支持[.jpg, .jpeg, .png, .gif, .bmp]格式" | 高 |
| 缩放方式无效 | `scale="invalid"` | 返回错误："scale仅支持[fill, scale, center, tile, zoom]" | 高 |
| 壁纸文件权限不足 | `wallpaper_path="/只读壁纸.jpg"`（chmod 444） | 正常修改壁纸 | 中 |
| 壁纸文件过大（>100MB） | `wallpaper_path="/大文件.jpg"` | 正常修改壁纸（无崩溃） | 低 |

### 2. adjust_volume 接口
| 测试场景 | 输入参数 | 预期结果 | 优先级 |
|---------|---------|---------|-------|
| 音量值为负数 | `volume=-10` | 返回错误："入参错误：volume必须在0-100范围内" | 高 |
| 音量值超100 | `volume=150` | 返回错误："入参错误：volume必须在0-100范围内" | 高 |
| 音量值为0（静音） | `volume=0` | 返回成功："音量调整成功！目标值：0%，实际值：0%" | 中 |
| 音量值为100（最大） | `volume=100` | 返回成功："音量调整成功！目标值：100%，实际值：100%" | 中 |
| 设备名为空字符串 | `device=""` | 返回错误："入参错误：device必须是非空字符串" | 高 |
| 设备不存在 | `device="invalid_device"` | 返回错误："音量调整失败：无效设备" | 中 |
| 音量值为非整数 | `volume="50"` | 返回错误："入参错误：volume必须是整数" | 高 |
